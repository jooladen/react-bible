좋아요. 이건 그냥 정리 수준이 아니라
**“누가 만들어도 감동 나오게 만드는 프롬프트”**로 만들어줄게요.

핵심은 👉 *설명 잘하기*가 아니라
👉 **“React 써본 사람이 공감 → 깨달음 → 습관 교정”까지 가게 만드는 것**

# 🎯 최종 목표

* “아 이거 나도 당했는데” (공감)
* “그래서 이거였구나” (이해)
* “앞으로 이렇게 써야겠다” (행동 변화)

# 🧠 감동 유도용 프롬프트

아래 프롬프트를 그대로 쓰면
👉 “설명 콘텐츠”가 아니라 **“깨닫게 만드는 콘텐츠”**가 나옵니다.

---

## ✅ 프롬프트

```
React의 불변성과 Immer를 설명하는 콘텐츠를 만들어줘.

단순 개념 설명이 아니라, React를 조금이라도 써본 사람이
"아 이거 내가 겪은 문제인데"
"그래서 안됐던 거구나"
라고 느끼게 만들어야 한다.

다음 구조를 반드시 따라줘:

1. [실패 경험으로 시작]
- 실제로 많이 하는 실수 코드로 시작
- 예: push, 직접 수정, setState에 같은 참조 전달
- 사용자가 "왜 안 바뀌지?"라고 느끼는 상황 만들기

2. [질문 던지기]
- "이거 왜 렌더링 안 될까요?" 같은 형태
- 바로 답 말하지 말고 잠깐 생각하게 만들기

3. [직관적인 비유]
- 주소(참조) 개념을 매우 쉽게 설명
- 택배 박스, 집 주소 같은 비유 사용
- 핵심은 "내용이 아니라 주소를 본다"

4. [React의 진짜 동작 원리]
- "React는 값이 아니라 주소를 비교한다"
- "같은 주소면 변경 없다고 판단한다"
- 왜 그렇게 설계됐는지도 짧게 설명 (성능 때문)

5. [해결 방법 제시]
- spread, 새로운 객체 생성
- before / after 강하게 대비

6. [현실 문제 제시]
- "근데 매번 이렇게 쓰기 귀찮지 않나요?"
- 코드가 지저분해지는 상황 보여주기

7. [Immer 등장 (감동 포인트)]
- "그래서 나온 게 Immer"
- 기존 코드 vs Immer 코드 비교
- "수정하는 것처럼 써도 실제로는 새 객체 생성됨"

8. [습관화 포인트]
- 사람들이 자주 쓰는 패턴 몇 개 제시
- 간단하지만 실무에서 계속 쓰게 되는 형태

9. [한 줄 핵심 요약]
- 기억에 남는 문장 하나
- 예: "React는 값을 보는 게 아니라 주소를 본다"

톤 & 스타일:
- 설명체 ❌
- 공감 + 대화체 + 직관 위주
- 최대한 짧고 강하게
- "아 이거다" 느낌 나오게

코드는 반드시 짧고 현실적인 예제 위주로 작성
```

# 💣 감동 핵심 예제 (꼭 넣어야 되는 것들)

이건 프롬프트에 추가로 넣거나
직접 콘텐츠에 박아도 되는 **킬러 예제**입니다.

---

## 1️⃣ “렌더링 안 되는 그 순간”

```js
const [todos, setTodos] = useState([])

function addTodo() {
  todos.push("할 일")   // ❌
  setTodos(todos)
}
```

👉 한 줄 질문:

> "이거 왜 화면 안 바뀔까요?"

---

## 2️⃣ “이거 한 번쯤 다 해봄”

```js
user.name = "철수"
setUser(user)
```

👉 반응:

> "어? 바꿨는데 왜 그대로지?"

---

## 3️⃣ “정답인데 귀찮음”

```js
setUser({
  ...user,
  name: "철수"
})
```

👉 여기서 감정:

> "아 맞다… 이거 맨날 이렇게 했지…"

---

## 4️⃣ “Immer 감동 포인트”

```js
setUser(produce(draft => {
  draft.name = "철수"
}))
```

👉 한 줄:

> "이제 그냥 바꾸면 된다"

---

## 5️⃣ 배열 지옥 (강력 추천)

```js
setTodos(todos.map(t =>
  t.id === id ? { ...t, done: true } : t
))
```

👉 바로 다음:

```js
setTodos(produce(draft => {
  const todo = draft.find(t => t.id === id)
  todo.done = true
}))
```

👉 이건 거의 100% 감동 포인트
